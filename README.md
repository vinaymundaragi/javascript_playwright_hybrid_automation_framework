📌 Overview
Hybrid UI + API automation framework built using Playwright + JavaScript.
This framework demonstrates:
  UI automation using Page Object Model
  API automation using Playwright request context
  Environment-based execution (dev/stage/prod)
  Retry & flaky test handling
  CI-ready configuration
  Parallel execution
  HTML reporting
🧱 Tech Stack
  Playwright
  JavaScript (ES6+)
  Node.js
  Page Object Model (POM)
  GitHub Actions (CI-ready)
  Faker API (for API testing demo)
